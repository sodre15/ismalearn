Combinação

A combinação é um conceito matemático utilizado na análise combinatória para contar a quantidade de subconjuntos possíveis que podem ser formados a partir de um conjunto maior, sem considerar a ordem dos elementos. Aqui estão algumas situações em que se usa a combinação: Escolha de Grupos: Quando você precisa formar grupos ou equipes a partir de um conjunto maior, e a ordem dos membros não importa..

Combinação Simples: C(n,p)=p!(n−p)!n!​ Onde: ( n ) é o número total de elementos. ( p ) é o número de elementos escolhidos. ( ! ) representa o fatorial de um número.

Combinação com repetição

A fórmula é C′(n,p)=p!(n−1)!(n+p−1)!​ onde: ( n ) é o número total de tipos de elementos. ( p ) é o número de elementos escolhidos

Permutação Simples

Permutação Simples: Utilizada quando todos os elementos de um conjunto são rearranjados. Fórmula: P(n)=n!

Permutação com Repetição

A fórmula da permutação com repetição é usada quando temos um conjunto de elementos onde alguns elementos se repetem.

Função do 1° Grau

A função do 1º grau, também conhecida como função afim, é uma função matemática que pode ser expressa na forma: f(x)=ax+b onde: (a) e (b) são números reais, (a) é diferente de zero.

Função do 2° grau

A função do 2º grau, também conhecida como função quadrática, é uma função polinomial que pode ser expressa na forma: f(x)=ax2+bx+c

Sistema de equação do 1° Grau

Existem vários métodos para resolver sistemas de equações do 1º grau, sendo os mais comuns: Método da Substituição: Isolamos uma das incógnitas em uma das equações. Substituímos essa expressão na outra equação. Resolvemos a equação resultante para encontrar o valor de uma das incógnitas. Substituímos esse valor na primeira equação para encontrar o valor da outra incógnita. Método da Adição (ou Eliminação): Manipulamos as equações para que uma das incógnitas tenha coeficientes opostos. Somamos as equações para eliminar uma das incógnitas. Resolvemos a equação resultante para encontrar o valor de uma das incógnitas. Substituímos esse valor em uma das equações originais para encontrar o valor da outra incógnita.

Juros Simples

Juros simples é um conceito básico em matemática financeira, onde os juros são calculados apenas sobre o valor inicial (capital) durante um período de tempo. A fórmula para calcular os juros simples é: J=C⋅i⋅t Onde: (J) é o valor dos juros, (C) é o capital inicial, (i) é a taxa de juros (em decimal), (t) é o tempo.

Juros compostos

Os juros compostos são calculados sobre o valor inicial (capital) e também sobre os juros acumulados de períodos anteriores, resultando em um crescimento exponencial. A fórmula para calcular os juros compostos é: M=C⋅(1+i)t Onde: (M) é o montante final, (C) é o capital inicial, (i) é a taxa de juros (em decimal), (t) é o tempo.

Logaritimo

O logaritmo é uma operação matemática que nos ajuda a resolver equações exponenciais. Ele é definido como o expoente ao qual devemos elevar uma base para obter um determinado número. A fórmula básica é: loga​b=x⟺ax=b

Frações

Uma fração é uma forma de representar partes de um todo. Ela é composta por dois números: o numerador (parte superior) e o denominador (parte inferior). O numerador indica quantas partes temos, enquanto o denominador mostra em quantas partes o todo foi dividido
