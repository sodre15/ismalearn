Hardware / Software

Exemplos
Componente Função
Processador (CPU) O cérebro do computador, responsável por executar instruções e processar dados.
Placa-mãe A placa principal que conecta todos os componentes do computador.
Memória RAM Armazena temporariamente os dados e programas em uso.
Armazenamento (HD/SSD) Dispositivo onde os dados são armazenados permanente
Placa de vídeo (GPU) Responsável pelo processamento gráfico e exibição de imagens.
Fonte de alimentação Fornece energia para todos os componentes do computador.
Gabinete Estrutura que abriga e protege todos os componentes internos.
Cooler Sistema de resfriamento para manter a temperatura dos componentes.
Unidades ópticas Leitores de CD/DVD/Blu-ray (opcional).
Periféricos Inclui monitor, teclado, mouse, etc.

Memórias

Exemplos
Tipo de memória Descrição
RAM (Random Access Memory) Memória volátil usada para armazenar dados temporários e programas em execução.
ROM (Read-Only Memory) Memória não volátil usada para armazenar firmware e dados permanentes.
Cache Memória de alta velocidade usada para armazenar dados frequentemente acessados pelo processador.
HD (Hard Disk) Dispositivo de armazenamento magnético usado para guardar grandes quantidades de dados.
SSD (Solid State Drive) Dispositivo de armazenamento em estado sólido, mais rápido e confiável que o HD.
Memória Flash Tipo de memória não volátil usada em dispositivos como pendrives e cartões de memória.
